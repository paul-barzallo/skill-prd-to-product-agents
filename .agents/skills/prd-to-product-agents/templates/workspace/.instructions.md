# Workspace Rules

This is a multi-agent product-development workspace with 9 agents.

**Full rule set:** `.github/copilot-instructions.md`
**Agent file rules:** `.github/instructions/agents.instructions.md`
**Documentation rules:** `.github/instructions/docs.instructions.md`

## Quick reference

- **Workspace overview:** `AGENTS.md`
- **Bootstrap summary:** `.state/bootstrap-report.md`
- **PRD entrypoint:** `bootstrap-from-prd` with `pm-orchestrator`
- **PRD clarity gate:** handled inside `bootstrap-from-prd` before planning or implementation continues
- **Source of truth:** Markdown/YAML files under `docs/project/`
- **Execution system:** GitHub Issues + GitHub Projects + Pull Requests
- **Branch model:** `develop` + task branches `<role>/<issue-id>-slug`
- **Final PR gate:** `devops-release-engineer`
- **GitHub governance contract:** `.github/github-governance.yaml`
- **Historical context:** Git (commits, PRs, issues, tags)
- **Audit ledger:** `.state/project_memory.db` -- passive, infrastructure-managed, agents do not interact
- **Capability contract:** `.github/workspace-capabilities.yaml` decides whether Git, GitHub automation, SQLite, markdownlint, or local-only mode are active
- **Local-only evidence:** `.state/local-history/` stores change records when Git capability is disabled
- **Runtime binary location:** `.agents/bin/prd-to-product-agents/` is the canonical location for `prdtp-agents-functions-cli`
- **Task closure:** `prdtp-agents-functions-cli git finalize` records the supported end-of-task evidence
- **Executive dashboard:** `docs/project/management-dashboard.md` refreshed by `prdtp-agents-functions-cli report dashboard`
- **Agent assembly:** run `prdtp-agents-functions-cli agents assemble` after editing `identity/` or `context/` source files
- **Model routing:** ordered `model:` fallback in agent frontmatter + prompt overrides for small/deep tasks

## Blocked-state format

When safe progress is not possible, respond with:

- `What happened`
- `Why blocked`
- `What input is needed`
- `Who should answer`
- `Next safe step`

## Instruction precedence

| Environment | Layers applied (highest to lowest) |
| ------------- | ----------------------------------- |
| **VS Code** | prompt frontmatter -> agent frontmatter -> `.github/instructions/*.instructions.md` -> `.github/copilot-instructions.md` -> root `.instructions.md` |
| **GitHub.com** | `.github/copilot-instructions.md` only (single-agent degraded mode) |
